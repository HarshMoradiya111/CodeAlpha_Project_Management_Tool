# CodeAlpha_Project_Management_Tool

Task 3 phase 1 scaffold for the CodeAlpha internship project management tool.

## Repo layout
- `backend/` - Express + MongoDB API
- `frontend/` - React + Vite client

## GitHub repository name
Use this exact repository name:

`CodeAlpha_Project_Management_Tool`

## Phase 1 scope
- Project setup
- Backend and frontend skeletons
- Base README and environment files

## Recommended setup
Backend:
1. `cd backend`
2. `npm install`
3. Copy `.env.example` to `.env`
4. Run `npm start`

Frontend:
1. `cd frontend`
2. `npm install`
3. Run `npm run dev`
